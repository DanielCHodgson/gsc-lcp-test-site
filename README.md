# Search Console Test Site (GitHub Pages)

This repository hosts a lightweight static website used to generate data inside **Google Search Console (GSC)** for testing integrations with the **SquaredUp Web API plugin**.

The site exists purely as a public property that Google can crawl and index so Search Console metrics can later be retrieved via the GSC API and surfaced inside dashboards.

---

## Purpose

This site is designed to:

- provide a verifiable Search Console property
- generate impressions / clicks / indexing signals
- support testing of GSC API queries
- simulate realistic page-level performance data
- act as a dataset source for a SquaredUp low-code plugin

It intentionally contains only static content.
