Dummy site for GSC testing

A lightweight website for testing search performance data

This simple GitHub Pages site exists so it can be verified in Google Search Console and used as a test property for API metrics.

View pages

What this site is for

This site is intentionally small, static, and easy to maintain. It gives Google Search Console a public property to collect impressions, clicks, queries, pages, and indexing data for.

Search analytics

Useful for testing clicks, impressions, CTR, average position, queries, and page-level performance.

Index coverage

Useful for checking whether pages are discovered, crawled, indexed, or excluded by Google.

Plugin testing

Once data appears in Search Console, it can be queried through the API and shaped into SquaredUp data streams.

Built as a lightweight GitHub Pages test site.
