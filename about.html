About the project

About this Search Console test site

This lightweight website is used to generate Google Search Console data for testing search analytics, indexing coverage, page performance, and API integrations.

Why this site exists

The site provides a simple public property that can be verified in Google Search Console. Once Google crawls and indexes the pages, the data can be queried through the Search Console API and shaped into SquaredUp Web API data streams.

Useful test signals

  • Search impressions by page
  • Search query performance
  • Click-through rate testing
  • Average position reporting
  • Indexed page coverage
