Search Console API documentation examples

This page describes example data streams that could be created from the Google Search Console Search Analytics API.

Example data streams

  • Top queries — query, clicks, impressions, CTR, position
  • Top pages — page URL, clicks, impressions, CTR, position
  • Performance over time — date, clicks, impressions
  • Device breakdown — device, clicks, impressions, CTR
  • Country breakdown — country, clicks, impressions, position

Example API dimensions

Useful Search Console dimensions include date, query, page, country, device, and search appearance. These dimensions can be combined to build dashboard tiles for organic search monitoring.
